Interface Rule

A Rule can include a RuleFunction property and must include an Assertions property. For each {@link RUle}, you can define only one RuleFunction. You can define one or more asserts within the Assertions property. If you don't define a RuleFunction, the Rule's Assertions always take effect.

see

https://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/rules-section-structure.html#rules-specific-intrinsic-section-structure
