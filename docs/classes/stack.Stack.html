    Parameters

      Logical ID of the PhysicalResource to delete.

    • state: UpdateState

    • allowedLogicalIds: Set<String>

      a set of logicalIds that are allowed to be deleted. This is so that we can delete a sub-set of the logicalIds when transiting dependencies, for example when deleting orphaned resources during a Stack Update.

Private evaluateParameter

  • evaluateParameter(state: UpdateState, paramName: string, paramDef: Parameter): Promise<Value>

  • Determine the value of a {@link paramName}.

    If the Parameter is a SSMParameterType then the value is fetched from AWS Systems Manager Parameter Store.

    The CloudFormationTemplate's Parameters and the input ParameterValues are assumed to be valid because the validateParameters function is called by updateStack.

    Parameters

    • state: UpdateState

      UpdateState being evaluated.

    • paramName: string

      name of the Parameter.

    • paramDef: Parameter

      the Parameter definition in the source CloudFormationTemplate.

    Returns Promise<Value>
